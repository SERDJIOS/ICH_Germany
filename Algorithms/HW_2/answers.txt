2. Задача коммивояжёра: сложность
Задача коммивояжёра — это задача поиска самого короткого маршрута, проходящего через все заданные города ровно по одному разу и возвращающегося в начальную точку.
Сложность: O(n!) — задача NP-трудная, на больших данных решается очень долго.

3. Временная сложность задач:

Task #1:
FOR i = 1 to n
  FOR j = 1 to n
    print "*"
    BREAK
Внутренний цикл выполняется один раз из-за BREAK.
Сложность: O(n)

Task #2:
FOR i = n/2 to n
  FOR j = 2 to n (j *= 2)
Внешний цикл: O(n)
Внутренний цикл: O(log n)
Сложность: O(n log n)

Task #3:
FOR i = 0 to n
  FOR j = i to 0
Суммарно выполняется как 1 + 2 + ... + n = O(n²)
Сложность: O(n²)

Task #4:
WHILE i > 0
  i = i / 2
Каждая итерация делит i пополам
Сложность: O(log n)
